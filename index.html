    The sort field. Default- results are sorted by "created date".

    created date

    The sort order if sort parameter is provided. One of asc or desc. Default- desc.

    desc

    The start Index from where the search results should be returned. Default - 0

    Example:

    0
  • maxItems: required (integer - default: 50)

    the number or Items that should be included in the results . Default - 50

    50
